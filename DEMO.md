# 使用演示指南

## 🎬 快速演示

### 場景：在電腦上管理手機照片

#### 步驟 1：啟動App
1. 在Android手機上開啟「MyApp」
2. 看到主畫面，顯示：
   - 📁 File Server 標題
   - ▶️ 開始服務按鈕（綠色）
   - ⏹️ 停止服務按鈕（灰色，不可用）
   - 狀態：已停止

#### 步驟 2：開始服務
1. 點擊「▶️ 開始服務」
2. 系統提示授權檔案存取權限
3. 點擊「允許」
4. Android 11+設備會提示：
   ```
   檔案權限
   請在設定中授予「所有檔案存取權限」以獲得最大檔案存取能力
   ```
5. 點擊「確定」
6. 手動前往：設定 → 應用程式 → MyApp → 權限 → 檔案和媒體
7. 選擇「允許管理所有檔案」
8. 返回App
9. 再次點擊「▶️ 開始服務」
10. 彈出對話框顯示：
    ```
    伺服器已啟動
    請在瀏覽器中開啟：
    http://192.168.1.100:8080
    ```
11. 狀態變為「運行中」，指示燈變綠

#### 步驟 3：連接電腦
1. 在同一Wi-Fi網路的電腦上
2. 開啟Chrome/Firefox/Edge瀏覽器
3. 輸入地址：`http://192.168.1.100:8080`
4. 看到檔案管理界面

#### 步驟 4：瀏覽照片
1. 看到根目錄的資料夾列表：
   ```
   📁 Download
   📁 DCIM
   📁 Pictures
   📁 Documents
   ...
   ```
2. 雙擊「📁 DCIM」資料夾
3. 雙擊「📁 Camera」資料夾
4. 看到所有手機拍攝的照片：
   ```
   📄 IMG_20260117_001.jpg  2.5 MB
   📄 IMG_20260117_002.jpg  3.1 MB
   📄 IMG_20260117_003.jpg  2.8 MB
   ...
   ```

#### 步驟 5：下載照片到電腦
1. 雙擊任一照片（例如 IMG_20260117_001.jpg）
2. 瀏覽器自動下載該照片到電腦
3. 查看手機App的日誌區域，新增一筆記錄：
   ```
   2026-01-17 15:30:25
   192.168.1.10
   GET
   /DCIM/Camera/IMG_20260117_001.jpg
   ```

#### 步驟 6：上傳檔案到手機
1. 在電腦上選擇要上傳的檔案（例如 document.pdf）
2. 拖曳到瀏覽器的「拖曳檔案到此處上傳」區域
3. 區域變綠，鬆開滑鼠
4. 檔案自動上傳
5. 檔案列表自動重新整理，看到新檔案
6. 手機日誌更新：
   ```
   2026-01-17 15:31:10
   192.168.1.10
   UPLOAD
   /DCIM/Camera/document.pdf
   ```

#### 步驟 7：整理檔案
1. 點擊「📁 新增資料夾」按鈕
2. 輸入資料夾名稱：`Backup`
3. 點擊「確定」
4. 新資料夾出現在列表中
5. 點擊檔案旁的「重新命名」按鈕
6. 輸入新名稱：`important_document.pdf`
7. 點擊「確定」
8. 檔案名稱更新
9. 手機日誌更新：
   ```
   2026-01-17 15:32:05
   192.168.1.10
   MKDIR
   /DCIM/Camera/Backup
   
   2026-01-17 15:32:30
   192.168.1.10
   RENAME
   /DCIM/Camera/document.pdf -> /DCIM/Camera/important_document.pdf
   ```

#### 步驟 8：刪除不需要的檔案
1. 選擇要刪除的檔案
2. 點擊「刪除」按鈕
3. 確認對話框：「確定要刪除嗎？」
4. 點擊「確定」
5. 檔案從列表中消失
6. 手機日誌記錄：
   ```
   2026-01-17 15:33:00
   192.168.1.10
   DELETE
   /DCIM/Camera/IMG_20260117_001.jpg
   ```

#### 步驟 9：查看歷史日誌
1. 在手機App上，看到日誌區域
2. 所有操作按時間倒序排列（最新在上）
3. 點擊日期選擇器
4. 選擇「2026-01-16」
5. 顯示昨天的日誌記錄
6. 可以切換不同日期查看歷史

#### 步驟 10：停止服務
1. 完成檔案管理後
2. 返回手機App
3. 點擊「⏹️ 停止服務」
4. 彈出確認：「伺服器已停止」
5. 狀態變為「已停止」，指示燈變灰
6. 電腦瀏覽器無法再連接

## 🎯 使用場景示例

### 場景 1：快速傳輸大檔案
**情況**：需要把電腦上的影片傳到手機
**步驟**：
1. 手機開啟服務
2. 電腦瀏覽器開啟網址
3. 進入 Download 資料夾
4. 拖曳影片檔案上傳
5. 完成後停止服務

**優點**：
- 無需USB線
- 速度快（區網傳輸）
- 操作簡單

### 場景 2：清理手機儲存空間
**情況**：手機空間不足，需要刪除大檔案
**步驟**：
1. 開啟服務
2. 電腦瀏覽器瀏覽各資料夾
3. 找出大檔案（顯示檔案大小）
4. 決定保留或刪除
5. 批次刪除不需要的檔案

**優點**：
- 大螢幕操作更方便
- 可以快速瀏覽
- 避免手機操作誤刪

### 場景 3：備份重要檔案
**情況**：定期備份手機文件到電腦
**步驟**：
1. 開啟服務
2. 進入 Documents 資料夾
3. 雙擊下載所有重要文件
4. 檢查日誌確認已下載
5. 停止服務

**優點**：
- 快速批次下載
- 有日誌記錄
- 可驗證完整性

### 場景 4：整理照片
**情況**：手機照片雜亂，需要分類整理
**步驟**：
1. 開啟服務
2. 進入 DCIM/Camera
3. 建立分類資料夾（旅遊、工作、家人）
4. 重新命名照片加上日期標記
5. 刪除模糊或重複照片

**優點**：
- 電腦鍵盤輸入快
- 大螢幕預覽方便
- 可使用電腦的圖片查看器

### 場景 5：團隊協作
**情況**：同事需要手機上的簡報檔案
**步驟**：
1. 開啟服務
2. 告知同事網址
3. 同事瀏覽器開啟
4. 下載所需檔案
5. 完成後停止服務

**優點**：
- 無需即時通訊軟體
- 不受檔案大小限制
- 同網路即可分享

## 💡 使用技巧

### 技巧 1：記住常用路徑
常用資料夾快速導航：
- 下載：`/Download`
- 相片：`/DCIM/Camera`
- 截圖：`/Pictures/Screenshots`
- 文件：`/Documents`
- 音樂：`/Music`

### 技巧 2：批次上傳
選擇多個檔案一次上傳：
1. 點擊「上傳檔案」按鈕
2. Ctrl+點擊選擇多個檔案
3. 或框選多個檔案
4. 確認上傳

### 技巧 3：快速搜尋
使用瀏覽器搜尋功能（Ctrl+F）：
1. 按 Ctrl+F
2. 輸入檔案名稱關鍵字
3. 快速定位檔案

### 技巧 4：保持服務運行
長時間使用時：
1. 設定手機「螢幕逾時」較長時間
2. 或連接充電器
3. 避免手機進入省電模式

### 技巧 5：查看IP變化
如果連接不上：
1. 檢查手機App顯示的IP是否改變
2. Wi-Fi重新連接可能改變IP
3. 使用新IP重新連接

## ⚠️ 注意事項

### 安全性提醒
1. ⚠️ 僅在信任的Wi-Fi網路使用
2. ⚠️ 使用完畢立即停止服務
3. ⚠️ 不要在公共場所開啟服務
4. ⚠️ 定期檢查日誌是否有異常存取

### 最佳實踐
1. ✅ 定期備份重要檔案
2. ✅ 上傳前檢查儲存空間
3. ✅ 刪除前再次確認
4. ✅ 使用有意義的檔案名稱
5. ✅ 定期清理舊日誌檔案

### 效能建議
1. 📊 單次上傳檔案建議 < 500MB
2. 📊 避免同時操作過多檔案
3. 📊 大檔案傳輸時保持網路穩定
4. 📊 定期重啟服務以釋放記憶體

## 🔧 疑難排解

### 問題：無法連接
**解決方案**：
1. 確認手機和電腦在同一Wi-Fi
2. 檢查手機App狀態是「運行中」
3. 確認輸入的IP和端口正確
4. 嘗試關閉電腦防火牆
5. 重啟服務

### 問題：上傳失敗
**解決方案**：
1. 檢查手機儲存空間
2. 確認資料夾有寫入權限
3. 檔案名稱避免特殊字元
4. 嘗試重新整理頁面

### 問題：找不到檔案
**解決方案**：
1. 確認檔案路徑正確
2. 檢查Android 11+權限設定
3. 某些系統資料夾可能無法存取
4. 嘗試從其他資料夾存取

### 問題：速度慢
**解決方案**：
1. 確認Wi-Fi訊號強度
2. 減少同時傳輸的檔案數量
3. 關閉其他佔用網路的應用
4. 考慮使用5GHz Wi-Fi頻段
