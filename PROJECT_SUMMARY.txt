🚀 快速開始
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣ 驗證項目
   $ bash verify.sh
   ✅ 檢查所有文件和環境

2️⃣ 本地構建（推薦）
   $ bash build.sh release
   ✅ APK 輸出：android/app/build/outputs/apk/release/app-release.apk

3️⃣ 或使用 Gradle
   $ cd android && ./gradlew assembleRelease
   ✅ APK 輸出同上

4️⃣ 安裝到設備
   $ adb install -r android/app/build/outputs/apk/release/app-release.apk

5️⃣ 運行應用
   - 打開 FileServeApp
   - 點擊「開始服務」
   - 訪問 http://<手機IP>:8080

🎯 API 端點
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

GET  /api/list?path=<path>        列出文件和目錄
GET  /api/get?path=<path>         下載文件
POST /api/upload                  上傳文件
DELETE /api/delete?path=<path>    刪除文件/目錄
POST /api/rename                  重命名文件
GET  /api/logs?date=<yyyyMMdd>    查詢日誌
